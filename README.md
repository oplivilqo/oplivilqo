# 魔法少女的魔女审判 文本框表情包生成器

一个自动化表情包生成工具，能够快速生成带有自定义文本的魔法少女的魔女裁判文本框图片。

是Python脚本[魔法少女的魔女裁判文本框脚本](https://github.com/oplivilqo/Text_box-of-mahoushoujo_no_majosaiban)的浏览器移植与实现。所有图片素材（连同上面的说明）都来自该项目。

![界面截图](screenshot.png)

## 使用方法

无需部署到Web服务器，直接下载整个仓库然后本地使用浏览器打开`index.html`即可使用。
